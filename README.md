# Monroe Public Library Government Links

## Overview
This repository contains a government resources website developed as part of my coursework while pursuing a Bachelor of Business Administration in Information Systems and Technology (IST).

The website provides organized access to government resources through categorized drop-down menus and demonstrates JavaScript event handling, form controls, dynamic navigation, and responsive webpage design. Users can select government agencies and public resources from multiple categories and automatically navigate to the selected website.

## Features
* Government resource directory
* Categorized government website listings
* Dynamic dropdown menu navigation
* Automatic website redirection
* Executive, Legislative, and Judicial branch resources
* State government resource directory
* Library-themed user interface
* Responsive webpage layout
